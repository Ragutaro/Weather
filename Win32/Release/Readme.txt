机上予報
Copyright 2017 ラグ太郎

机上予報は、Yahoo!天気・災害(https://weather.yahoo.co.jp/weather/)から天気のデータを取得し、見やすく表示することを目的とするソフトウェアです。
kawai氏製作の「タスクトレイに天気予報」が正常に動作しなくなったため、自分用として製作したものですが、
なかなか良い感じに出来上がったので公開することとしました。

このような経緯のため「タスクトレイに天気予報」と画面構成などが似ていますが、幾つかの点で改良を加えており、
より使いやすくなっていると思います。

改良点
	・「今日の天気」と「明日の天気」の全てを表示できるようにした。
	・降水量や風向きも表示するようにした。
	・注意報や警報を表示するようにした。
	・雨雲の動きを表示できるようにした。

【使い方】
初めての起動時には、東京都千代田区の天気が表示されます。机上予報上で右クリックするとメニューが出るので、
[地域の選択]を選んでご自宅の地域を設定してください。

机上予報の画面中央辺りにマウスカーソルを移動すると、ポップアップでコントロールバーが出ます。
このコントロールバーで、「今日の天気」と「明日の天気」の全てをスクロールで表示することが出来ます。

	・左矢印をクリックすると、前の時間帯の天気を表示します。
	・リロードをクリックすると、現在の時間帯の天気を表示します。
	・右矢印をクリックすると、次の時間帯の転機を表示します。

【雨雲の動き】
メニューの[雨雲の動き]を選ぶと、Yahoo!天気・災害から雨雲の動きのGIF画像をダウンロードし、別ウィンドウで表示します。
このアニメーションGIFは三回繰り返すと停止します。停止したときは、画像をクリックすると再度アニメーションを開始します。

【カスタマイズ】
カスタマイズできるのは、背景画像、天気アイコン、フォントの種類・サイズ・色のみです。画面のレイアウトを変更することは出来ません。
